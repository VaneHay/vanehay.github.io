docker 运行时php5.x 环境是可以的

不过你需要把你在本地项目的目录映射到 docker php5.x的容器里面
docker pull php:5.3

至于你说的 tp3.2 连 mysql 可以把宿主机内网ip地址加上3306端口映射到容器中 。
但这并不是一个好办法

正确的是在docker内创建一个 mysql 然后设置 php 和 mysql两个处于同一网段即可。

就像上面的兄弟说的，用 docker-compose 做可以更方便。
