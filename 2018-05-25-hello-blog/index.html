Hello Blog

May 25, 2018 | other

自工作起也有好几年啦，工作中遇到问题总习惯性地Google、百度搜索，待问题解决随手就会把资料页关掉，造成一个不好的习惯就是下次遇到问题，还会习惯性地去搜索引擎找解决办法，这种习惯导致对搜索引擎的依赖相当严重。

后来总会 Crtl+D 保存成书签，那是还没接触到 Chrome 书签同步功能，每次重装电脑书签随之也丢掉了。

再后来接触过各种免费的博客系统，和讯博客（当时唯一一个支持挂广告联盟代码的博客系统）、百度博客，csdn、wordpress……等，但都没能坚持去记录自己的点点滴滴。

后来从 debian 完美过度到 ubuntu 上发现 ubuntu 自带的文本编辑框（Gedit）再强大的图形界面下视觉体验相当棒，索性就拿 Gedit 写笔记，保存成文本放电脑文件夹内。

这种体验延续了好久，直到某天 markdown 的兴起，视觉体验又上一个高度，而且感觉挺酷的。研究完怎么在 Github ＋ hexo 搭建免费博客，随后又被抛到九霄云外，继续使用 text 记录笔记。

…

最近重读一遍 markdown 文档，发现各大主流 git 衍生的版本库对 markdown 的支持相当棒，至少视觉上挺舒服，所以搭建起免费的博客，准备进行笔记大迁移，一方面自己对笔记的温习回顾，另外一方面或许能帮到别人。

就写到这吧… 关注我，我是 VaneHay
